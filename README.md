## ✨ Features at a Glance

### 🐧 Linux Environment (Chroot)
Turn your Android into a powerhouse. Download and install a full Arch Linux rootfs directly from the app. 
- **1-Click Install:** Download a 1.75GB optimized Arch Linux image straight to your device.
- **VNC & Web Access:** Launch a graphical desktop environment (GUI) or connect via Web VNC directly from your phone.
- **Terminal Integration:** Drop into a root shell or access the full Arch Linux terminal seamlessly.

### 📜 Dynamic Scripting Engine
Say goodbye to rigid apps. GamerX Manager utilizes a dynamic scripting system (Scripts V9).
- **Import custom scripts** via `.zip` files.
- **Auto-generated UI:** The app reads the script's `manifest.json` and automatically builds beautiful inputs, sliders, and toggles for you.
- **Quick Settings Integration:** Execute your favorite scripts directly from your Android drop-down Quick Settings tiles.

### 🎨 Systemless Customization
Personalize every aspect of your phone without touching the system partition.
- **Boot Animations:** Easily import and apply custom boot animations.
- **Icon Packs:** Systemlessly apply stunning icon packs across your device.
- **Ringtones & UI Sounds:** Change your system sounds easily.

## 🚀 Getting Started

1. **Install the App:** Download the latest `GamerXManager.apk` from the [Releases](https://github.com/GamerX3560/GamerXManager/releases) page.
2. **Flash the Module:** Download the `gamerx_module_v11.37_clean.zip` (or newer) from the Releases page and flash it via **KernelSU** or **Magisk**.
3. **Grant Root:** Open GamerX Manager and grant it root/superuser permissions when prompted.
4. **Explore & Customize:** Import scripts, tweak settings, or download the Linux environment!
